The BMAD Master Agent (`@bmad-master`) is your primary guide for structured, expert-led development of XtremFlow using the BMad Method Framework. It orchestrates all specialized agents (PM, Architect, Developer, QA, Security Expert) through proven workflows.

- Recommends prioritization

### 💡 Intelligence Features

## Workflows Managed by BMAD Master

### Workflow 1: Implement Feature (New)
```
Phase 1: Analysis
  → @analyst examines current state
  → Creates analysis document

Phase 2: Planning
  → @pm creates user stories
  → Estimates effort
  → Creates implementation roadmap

Phase 3: Solutioning
  → @architect designs solution
  → Reviews trade-offs
  → Creates technical specifications

Phase 4: Implementation
  → @developer writes code
  → Follows design specifications
  → Creates production-ready implementation

Phase 5: Quality Assurance
  → @qa designs test strategy
  → Creates comprehensive tests
  → Validates coverage (target: 80%+)

Phase 6: Review
  → @code-reviewer reviews implementation
  → Ensures alignment with design
  → Provides feedback for improvements
```

### Workflow 2: Fix Critical Issue (Security-First)
```
Phase 1: Analysis
  → @security-expert analyzes issue
  → Identifies vulnerabilities
  → Assesses impact

Phase 2: Design
  → @architect designs secure solution
  → @security-expert validates approach
  
Phase 3: Implementation
  → @developer codes with security controls
  → @security-expert reviews code
  
Phase 4: Testing
  → @qa creates security test cases
  → @security-expert validates
```

### Workflow 3: Rapid Feature (Quick Path)
```
For less critical items:
1. @architect → Quick design sketch
2. @developer → Implementation
3. @qa → Basic tests
4. @reviewer → Code review
```

### Workflow 4: Weekly Planning
```
1. Review roadmap progress
2. Identify what's blockers
3. Plan daily sprints
4. Assign agents to each task
5. Create artifact outputs
```
