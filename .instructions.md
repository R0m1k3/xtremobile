## 🔄 BMad Method Phases Explained

### 1️⃣ **Analysis** (`_bmad/bmm/1-analysis/`)
- Understand current state
- Identify problems
- Gather requirements
- **Output**: Analysis documents

### 2️⃣ **Planning** (`_bmad/bmm/1-analysis/`)
- Define user stories
- Create roadmap
- Estimate effort
- **Output**: Planning artifacts

### 3️⃣ **Solutioning** (`_bmad/bmm/3-solutioning/`)
- Design architecture
- Evaluate options
- Create technical specs
- **Output**: Solution designs

### 4️⃣ **Implementation** (`_bmad/bmm/4-implementation/`)
- Write code
- Create tests
- Document features
- **Output**: Code + tests
